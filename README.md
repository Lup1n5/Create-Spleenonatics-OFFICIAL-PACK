A heavily modded NeoForge 1.21.1 modpack centred around Create and Create: Aeronautics, featuring exploration, automation, transport, and aerial combat.

## Installation

### Modrinth App (recommended)
1. Download the `.mrpack` file from the latest release
2. Open the Modrinth App
3. Click **Add Instance** → **Import from file**
4. Select the `.mrpack` and let it install

### Other launchers (Prism, ATLauncher, CurseForge, etc.)
Most modern launchers support importing `.mrpack` files directly:
- **Prism Launcher:** Add Instance → Import → select the `.mrpack`
- **ATLauncher:** Instances → Import → select the `.mrpack`
- **CurseForge App:** Create Custom Profile → Import → select the `.mrpack`

If your launcher doesn't support `.mrpack`, use the `mods.zip` instead — see below.

### Manual install using mods.zip
1. Install [NeoForge 21.1.230](https://neoforged.net) for Minecraft 1.21.1
2. Download and extract `mods.zip`
3. Copy the `mods` folder into your Minecraft instance directory, replacing the existing one
4. Launch the game

## Updating while keeping your keybinds and settings

If you already have a Modrinth instance set up and want to update without losing your keybinds, options, or shader settings, do this instead of reimporting the pack:

1. Download the new [mods.zip](https://drive.google.com/file/d/1gs4HMC9lrSVLpal8R-3Xhk2hTcMkpVWt/view?usp=sharing) from the link release
2. Extract the zip — you'll get a `mods` folder
3. Open your existing Modrinth instance folder:
   - In the Modrinth App, right-click your instance → **Open Folder**
4. Delete the existing `mods` folder inside it
5. Copy the new `mods` folder in its place
6. Launch as normal — your `options.txt`, keybinds, shader settings, and configs are all untouched
